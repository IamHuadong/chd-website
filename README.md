 
# chd-website
## 网站预览： [个人网站地址](http://chd1994.club:8086)
介绍：vue的多页面应用搭建的个人网站，将之前的项目单独做个页面加入进去  
（目前发现vue的多页面配置比较难更新，每加一个小项目进去就要运行整个项目。不如用vue的单页面应用，在后端多写个路由，后期加入新项目时，网站调试和更新很方便。）
